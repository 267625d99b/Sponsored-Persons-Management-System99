## 🛠️ السكريبتات

### سكريبتات النشر

| السكريبت | الوصف | الاستخدام | الأولوية |
|----------|-------|-----------|----------|
| `deploy-free.sh` | تحضير للنشر المجاني | `./deploy-free.sh` | ⭐⭐⭐⭐⭐ |
| `check-deployment.sh` | فحص جاهزية النشر | `./check-deployment.sh` | ⭐⭐⭐⭐ |
| `deploy-docker.sh` | نشر باستخدام Docker | `./deploy-docker.sh` | ⭐⭐⭐⭐ |
| `deploy-vps.sh` | نشر تلقائي على VPS | `./deploy-vps.sh` | ⭐⭐⭐⭐ |
| `update-deployment.sh` | تحديث النشر | `./update-deployment.sh` | ⭐⭐⭐ |
| `deploy.sh` | نشر يدوي بسيط | `./deploy.sh` | ⭐⭐ |

### أوامر npm

```bash
# تثبيت جميع الحزم
npm run install:all

# تشغيل للتطوير
npm run dev

# بناء Frontend
npm run build:frontend

# فحص الجاهزية
npm run deploy:check

# توليد مفاتيح آمنة
npm run generate:keys

# Docker
npm run docker:up
npm run docker:down
npm run docker:logs
```

### أوامر Makefile

```bash
# عرض جميع الأوامر
make help

# التثبيت
make install

# التطوير
make dev

# البناء
make build

# النشر
make deploy-docker
make deploy-vps

# Docker
make docker-up
make docker-down
make docker-logs

# أدوات
make keys
make clean
```

## 🎯 خارطة الطريق

### 1. للمبتدئين (نشر مجاني)

```
1. اقرأ: نشر_مجاني.md
   ↓
2. نفذ: ./deploy-free.sh
   ↓
3. ارفع على GitHub
   ↓
4. انشر على Render
   ↓
5. استمتع! 🎉
```

### 2. للمتقدمين (VPS)

```
1. اقرأ: دليل_النشر.md
   ↓
2. نفذ: ./deploy-vps.sh
   ↓
3. أدخل معلومات السيرفر
   ↓
4. انتظر (تلقائي)
   ↓
5. استمتع! 🎉
```

### 3. للتطوير المحلي

```
1. اقرأ: QUICK_START.md
   ↓
2. نفذ: npm run install:all
   ↓
3. نفذ: npm run dev
   ↓
4. افتح: http://localhost:3000
   ↓
5. ابدأ التطوير! 💻
```
