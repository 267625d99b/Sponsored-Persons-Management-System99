## 🛠️ الأوامر المفيدة

### npm

```bash
# تثبيت
npm run install:all

# تطوير
npm run dev

# بناء
npm run build:frontend

# فحص
npm run deploy:check

# مفاتيح
npm run generate:keys

# Docker
npm run docker:up
npm run docker:down
```

### Makefile

```bash
# عرض الأوامر
make help

# تثبيت
make install

# تطوير
make dev

# نشر
make deploy-docker
make deploy-vps

# مفاتيح
make keys
```
