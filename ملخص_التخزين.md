## 🗂️ هيكل قاعدة البيانات

### الجداول الموجودة:

#### 1. admins (المدراء)
```sql
- id: رقم تعريفي
- email: البريد الإلكتروني
- password: كلمة المرور (مشفرة)
- name: الاسم
- created_at: تاريخ الإنشاء
```

#### 2. sponsored (المكفولين)
```sql
- id: رقم تعريفي
- full_name: الاسم الكامل
- id_number: رقم الهوية
- phone: رقم الهاتف
- sponsorship_start_date: تاريخ بداية الكفالة
- annual_amount: المبلغ السنوي
- status: الحالة (نشط/متوقف)
- notes: ملاحظات
- created_at: تاريخ الإنشاء
- updated_at: تاريخ التحديث
```

#### 3. payments (المدفوعات)
```sql
- id: رقم تعريفي
- sponsored_id: رقم المكفول
- amount: المبلغ
- payment_date: تاريخ الدفع
- year: السنة
- notes: ملاحظات
- created_at: تاريخ الإنشاء
```

#### 4. documents (المستندات)
```sql
- id: رقم تعريفي
- sponsored_id: رقم المكفول
- filename: اسم الملف
- original_name: الاسم الأصلي
- file_type: نوع الملف
- file_size: حجم الملف
- category: التصنيف
- description: الوصف
- created_at: تاريخ الإنشاء
```

#### 5. activity_logs (سجل النشاطات)
```sql
- id: رقم تعريفي
- admin_id: رقم المدير
- admin_name: اسم المدير
- action: العملية
- entity_type: نوع الكيان
- entity_id: رقم الكيان
- entity_name: اسم الكيان
- details: التفاصيل
- ip_address: عنوان IP
- created_at: تاريخ الإنشاء
```

#### 6. notifications (الإشعارات)
```sql
- id: رقم تعريفي
- title: العنوان
- message: الرسالة
- type: النوع
- is_read: مقروء/غير مقروء
- created_at: تاريخ الإنشاء
```

#### 7. settings (الإعدادات)
```sql
- id: رقم تعريفي (دائماً 1)
- currency: العملة
- currency_symbol: رمز العملة
- date_format: تنسيق التاريخ
- renewal_reminder_days: أيام التذكير
- enable_2fa: تفعيل 2FA
- organization_name: اسم المؤسسة
- email_service: خدمة البريد
- email_user: بريد المرسل
- email_pass: كلمة مرور البريد
- updated_at: تاريخ التحديث
```

#### 8. backups (النسخ الاحتياطية)
```sql
- id: رقم تعريفي
- filename: اسم الملف
- file_size: حجم الملف
- type: النوع (يدوي/تلقائي)
- created_at: تاريخ الإنشاء
```

## 🔄 النسخ الاحتياطي والاستعادة

### إنشاء نسخة احتياطية يدوية:
```
1. اذهب إلى الإعدادات
2. تبويب "النسخ الاحتياطي"
3. اضغط "إنشاء نسخة احتياطية"
4. يتم حفظها في backend/backups/
```

### استعادة نسخة احتياطية:
```
1. اذهب إلى الإعدادات
2. تبويب "النسخ الاحتياطي"
3. اختر النسخة المطلوبة
4. اضغط "استعادة"
5. حدّث الصفحة
```

### تحميل نسخة احتياطية:
```
1. اذهب إلى الإعدادات
2. تبويب "النسخ الاحتياطي"
3. اضغط "تحميل" بجانب النسخة
4. يتم تحميلها على جهازك
```
