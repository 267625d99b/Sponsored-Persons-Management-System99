# دليل إدارة المدراء

## التحديثات الجديدة

تم إضافة نظام كامل لإدارة حسابات المدراء في النظام.

## الميزات الجديدة

### 1. إنشاء حسابات متعددة
- يمكن الآن إنشاء أكثر من حساب مدير
- كل مدير له صلاحيات كاملة على النظام

### 2. واجهة إدارة المدراء
- تبويب جديد في صفحة الإعدادات: **إدارة المدراء**
- عرض قائمة بجميع المدراء
- إضافة مدير جديد
- تعديل معلومات المدير (الاسم والبريد)
- حذف المدراء (لا يمكن حذف نفسك)

### 3. API Endpoints الجديدة

#### إنشاء مدير جديد (محمي)
```
POST /api/auth/create-admin
Authorization: Bearer <token>
Body: {
  "email": "admin@example.com",
  "password": "password123",
  "name": "اسم المدير"
}
```

#### الحصول على قائمة المدراء
```
GET /api/auth/admins
Authorization: Bearer <token>
```

#### تحديث معلومات مدير
```
PUT /api/auth/admins/:id
Authorization: Bearer <token>
Body: {
  "name": "الاسم الجديد",
  "email": "email@example.com"
}
```

#### حذف مدير
```
DELETE /api/auth/admins/:id
Authorization: Bearer <token>
```

#### تغيير كلمة المرور
```
POST /api/auth/change-password
Authorization: Bearer <token>
Body: {
  "currentPassword": "كلمة المرور الحالية",
  "newPassword": "كلمة المرور الجديدة"
}
```

## كيفية الاستخدام

### 1. الوصول إلى إدارة المدراء
1. سجل دخول إلى النظام
2. اذهب إلى **الإعدادات**
3. اختر تبويب **إدارة المدراء**

### 2. إضافة مدير جديد
1. اضغط على زر **إضافة مدير**
2. أدخل البيانات:
   - الاسم
   - البريد الإلكتروني
   - كلمة المرور (6 أحرف على الأقل)
3. اضغط **إنشاء المدير**

### 3. تعديل مدير
1. اضغط على زر **تعديل** بجانب المدير
2. عدّل الاسم أو البريد
3. اضغط **حفظ التغييرات**

### 4. حذف مدير
1. اضغط على زر **حذف** بجانب المدير
2. أكد الحذف
3. **ملاحظة:** لا يمكنك حذف حسابك الخاص

## الأمان

- جميع endpoints محمية بـ JWT Token
- لا يمكن للمدير حذف نفسه
- يتم تسجيل جميع العمليات في سجل النشاطات
- كلمات المرور مشفرة باستخدام bcrypt

## الحساب الافتراضي

عند أول تشغيل للنظام، يتم إنشاء حساب مدير افتراضي:

```
البريد: admin@example.com
كلمة المرور: Admin@123
```

**مهم:** غيّر كلمة المرور فوراً بعد أول تسجيل دخول!
