# 🚀 دليل النشر السريع

## خيارات النشر المتاحة

### 1️⃣ النشر باستخدام Docker (الأسهل) ⭐

```bash
# تأكد من تثبيت Docker و Docker Compose
chmod +x deploy-docker.sh
./deploy-docker.sh
```

الموقع سيكون متاح على: http://localhost

---

### 2️⃣ النشر على VPS (موصى به للإنتاج)

```bash
# على جهازك المحلي
chmod +x deploy-vps.sh
./deploy-vps.sh
```

السكريبت سيقوم بـ:
- ✅ إعداد السيرفر تلقائياً
- ✅ تثبيت جميع المتطلبات
- ✅ توليد مفاتيح آمنة
- ✅ الحصول على شهادة SSL
- ✅ تشغيل النظام

---

### 3️⃣ النشر اليدوي

راجع ملف `دليل_النشر.md` للتعليمات التفصيلية.

---

## 🔐 بيانات الدخول الافتراضية

- **البريد**: admin@example.com
- **كلمة المرور**: Admin@123

⚠️ **مهم جداً**: غيّر كلمة المرور فوراً بعد أول تسجيل دخول!

---

## 📋 قائمة التحقق قبل النشر

- [ ] تثبيت Node.js 18+
- [ ] تثبيت npm
- [ ] (للـ Docker) تثبيت Docker و Docker Compose
- [ ] (للـ VPS) الحصول على نطاق (domain)
- [ ] إعداد بريد إلكتروني للـ 2FA

---

## 🆘 المساعدة

إذا واجهت مشاكل:

1. راجع ملف `دليل_حل_المشاكل.md`
2. افحص السجلات:
   - Docker: `docker-compose logs -f`
   - VPS: `ssh root@your-server 'pm2 logs'`
