## 📝 الخطوات التالية (10 دقائق)

### الخطوة 1: رفع المشروع على GitHub (3 دقائق)

```bash
# 1. إنشاء Git repository (إذا لم يكن موجوداً)
git init
git add .
git commit -m "Ready for deployment"

# 2. اذهب إلى https://github.com/new
# 3. أنشئ repository جديد (اسمه مثلاً: sponsorship-system)
# 4. لا تضف README أو .gitignore أو license

# 5. ارجع للـ terminal ونفذ:
git remote add origin https://github.com/YOUR_USERNAME/sponsorship-system.git
git branch -M main
git push -u origin main
```

---

### الخطوة 2: النشر على Render (7 دقائق)

#### أ. إنشاء حساب (دقيقة واحدة)

1. اذهب إلى: https://render.com
2. اضغط **Get Started for Free**
3. سجل دخول بحساب GitHub
4. اسمح لـ Render بالوصول للـ repository

#### ب. نشر Backend (3 دقائق)

1. من لوحة التحكم، اضغط **New +** → **Web Service**
2. اختر المشروع من القائمة
3. املأ البيانات:

```
Name: sponsorship-backend
Region: Frankfurt (أو الأقرب لك)
Branch: main
Root Directory: backend
Runtime: Node
Build Command: npm install
Start Command: node src/server.js
Instance Type: Free
```

4. اضغط **Advanced** وأضف المتغيرات البيئية:

```
NODE_ENV = production
PORT = 10000
JWT_SECRET = 12b87dba030e498a05c31e0c40b9f943bd256b28b93affd543c11b30d44f5554cbd7aac09099f0cdb4d1dfe4741b2fe73c97de7d2e3f101309e933006bb31fb9
ENCRYPTION_KEY = 41f6027b1a56a3f992c28dbf4056766cdea6dda15f1f888c6a76b298d3a62f68
EMAIL_SERVICE = gmail
EMAIL_USER = your_email@gmail.com
EMAIL_PASS = your_app_password_from_gmail
CORS_ORIGIN = https://sponsorship-frontend.onrender.com
```

5. اضغط **Create Web Service**
6. انتظر 5-10 دقائق
7. **انسخ الـ URL** (مثلاً: `https://sponsorship-backend-abc123.onrender.com`)

#### ج. نشر Frontend (3 دقائق)

1. من لوحة التحكم، اضغط **New +** → **Static Site**
2. اختر نفس المشروع
3. املأ البيانات:

```
Name: sponsorship-frontend
Region: Frankfurt
Branch: main
Root Directory: frontend
Build Command: npm install && npm run build
Publish Directory: dist
```

4. أضف متغير بيئي واحد:

```
VITE_API_URL = [الصق URL الـ Backend من الخطوة السابقة]
```

5. اضغط **Create Static Site**
6. انتظر 3-5 دقائق

#### د. تحديث CORS (دقيقة واحدة)

1. ارجع لـ **Backend** في Render
2. اذهب إلى **Environment**
3. عدّل `CORS_ORIGIN` إلى URL الـ Frontend الفعلي
4. احفظ (سيعيد النشر تلقائياً)

## 🆘 إذا واجهت مشاكل

### المشكلة: Build Failed على Render

**الحل**: افحص السجلات (Logs) في Render وتأكد من:
- Root Directory صحيح
- Build Command صحيح
- جميع المتغيرات البيئية موجودة

### المشكلة: CORS Error

**الحل**: تأكد من `CORS_ORIGIN` في Backend يطابق URL الـ Frontend بالضبط

### المشكلة: Backend بطيء

**الحل**: هذا طبيعي في الخطة المجانية. استخدم UptimeRobot
