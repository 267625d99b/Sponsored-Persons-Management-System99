## 🔐 المتغيرات البيئية المطلوبة

### للنشر المجاني

```env
NODE_ENV=production
PORT=10000
JWT_SECRET=<ولّد مفتاح 64 حرف>
ENCRYPTION_KEY=<ولّد مفتاح 32 حرف>
EMAIL_SERVICE=gmail
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=<App Password من Gmail>
CORS_ORIGIN=https://your-frontend-url.com
```

### توليد المفاتيح

```bash
# طريقة 1
npm run generate:keys

# طريقة 2
make keys

# طريقة 3
./deploy-free.sh  # يولدها تلقائياً
```

**الدليل الكامل:** `ENVIRONMENT_VARIABLES.md`

## 📱 بعد النشر

### 1. تسجيل الدخول

```
البريد: admin@example.com
كلمة المرور: Admin@123
```

⚠️ **غيّر كلمة المرور فوراً!**

### 2. إعداد البريد الإلكتروني

اتبع التعليمات في `نشر_مجاني.md` قسم "إعداد Gmail"

### 3. منع التوقف (للنشر المجاني)

استخدم [UptimeRobot](https://uptimerobot.com) - مجاني

### 4. النسخ الاحتياطي

من لوحة التحكم: **الإعدادات** → **النسخ الاحتياطي**

## 🆘 حل المشاكل

### المشكلة: لا أعرف من أين أبدأ

**الحل:** اتبع `نشر_مجاني.md` خطوة بخطوة

### المشكلة: Backend بطيء

**الحل:** استخدم UptimeRobot (شرح في `نشر_مجاني.md`)

### المشكلة: CORS Error

**الحل:** تأكد من `CORS_ORIGIN` صحيح (شرح في `ENVIRONMENT_VARIABLES.md`)

### المشكلة: البريد لا يعمل

**الحل:** استخدم App Password من Gmail (شرح في `نشر_مجاني.md`)

**الدليل الكامل:** `دليل_حل_المشاكل.md`
